# Java Script 

## Expresions

An expression assigns/evaluates variables.

`var color= beige;`
Set the variable to the "value" beige.

`var area = 3*2;`
Sets the variable area to (3x2) or 6.

## Operators

Operators are what allow programmers to merge two values into one.

`var area = 3*2;`

The operator is:
`3*2`
